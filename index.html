Pulzine

Small, fast 2D game engine made by single person.

Pulzine is a small(for now) game engine made in C++ with C# and C++ as scripting languages and graphical editor. Currently, only Windows is supported, Mac and Linux support is coming soon.

Features
  • 2D Sprites + spritesheets rendering
  • ECS
  • Scriptable language: C++
  • Math library
